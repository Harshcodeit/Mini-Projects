### ❤️ Heart Disease Prediction

A machine learning project that predicts whether a person is likely to have heart disease based on medical attributes.

- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- Resting ECG
- Max Heart Rate
- Exercise-Induced Angina
- Oldpeak
- ST Slope

**Tools & Libraries**

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Streamlit
